Ricerca di un libro

Per fare la ricerca di un libro può essere scelta una colonna della tabella per fare l'ordinamento,
oppure si può fare una ricerca usando uno dei seguenti metadati che possano rappresentare un libro:

  • Titolo
  • Autore
  • Argomento
  • Anno

Prenotazione di un libro

Per prenotare un libro cliccare il bottone "Prenota libro", ed inserire nome e cognome
